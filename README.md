# CyberPulse

CyberPulse is a full-stack monitoring system that collects system metrics, processes them through a FastAPI backend, and displays them on a neon cyber-Y2K dashboard with real-time charts, alerts, and lightweight AI for anomaly detection.

## Key Features

- **Data Collection:** Bash scripts (CPU, RAM, disk, network, temperature, logs).
- **Backend:** FastAPI Python API `/metrics`, `/metrics/latest`, `/metrics/history`, `/alerts`, `/predict`.
- **Database:** SQL for historical metrics and logs.
- **AI:** Simple anomaly detection and usage prediction.
- **Frontend:** Futuristic monitoring dashboard with metrics, charts, and alert panel.
- **Cloud Deployment:** Deployable on AWS, Azure, GCP, Render, Railway, Netlify, Vercel.
- **Security:** HTTPS, tokens, CORS.
- **Documentation:** README, API docs, diagrams, guides.
- **Tests:** Unit, integration, and load tests.

## Tech Stack

- Frontend: HTML, CSS, JavaScript, Chart.js
- Backend: Python, FastAPI
- Database: SQL (PostgreSQL/MySQL) or MongoDB
- Scripts: Bash (optional C/Assembly)
- Cloud: AWS, Azure, GCP, Netlify, Vercel
- AI: Python (scikit-learn or similar)

## Design Style

- Dark/black background, gray details
- Neon green, purple, and tech-Y2K accents
- Retro-futuristic, sci-fi 2000 style
- Optional ASCII “hacker terminal” mode

## Project Structure

- /frontend – dashboard, pages, scripts, assets
- /backend – API, models, services, database connectors
- /scripts – Bash collectors, cronjobs
- /db – schemas, queries, exports
- /docs – architecture, API, installation, roadmap
- /art – ASCII logo, mockups, icons, wallpapers
- /tests – unit, integration, load tests


## Goal

Build a professional portfolio project that demonstrates DevOps, Cloud, Full-Stack Development, Bash scripting, Database design, Security, AI integration, and futuristic UI/UX design.
